# 🚀 AI Full Stack Task Manager (Flask + AI + WhatsApp API)

## 📌 Project Overview
This is a full stack AI-powered task management system built using Flask, MySQL, and AI APIs. It includes WhatsApp notifications and AI automation features.

---

## ⚙️ Tech Stack
- Python (Flask)
- MySQL + SQLAlchemy
- HTML, CSS, JavaScript
- OpenAI / Gemini API
- WhatsApp API (Twilio)
- Docker
- AWS (EC2, S3)

---

## 🚀 Features
- User authentication system
- Task CRUD operations
- AI-powered task suggestions
- WhatsApp notifications (individual + group)
- REST API backend
- Dockerized deployment
